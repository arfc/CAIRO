# README for Data


### UIUC Data

The data included in this folder related to UIUC is
* railsplitter_data.csv
* lincoln_weatherdata.csv
* champaig_nweatherdata.csv
* uiuc_demand_data.csv
* solarfarm_data.csv


### Other Data

* Columns:
    - Pressure
    - Temperature
    - DHI
    - DNI
    - GHI
    - Albedo
    - Relative Humidity
    - Wind Speed

* Irradiance data for: 
    - San Bernadino County, CA (specifically, Barstow, CA)
    - Santa Fe, NM
    - Dallas, TX
    - Champaign, IL
    - Mansfield, OH
    - Syracuse, NY


* Wind Speed data for (at 100m height):
    - Barstow, CA
    - Santa Fe, NM
    - Dallas, TX
    - Lincoln, IL
    - Mansfield, OH
    - Syracuse, NY
